# zhihua的个人博客

记录技术与生活。

基于 [Hexo](https://hexo.io/) 搭建，使用 Landscape 主题，部署在 GitHub Pages。

## 快速开始

```bash
# 安装依赖
npm install

# 本地预览
npm run server

# 生成静态文件
npm run build

# 部署
npm run deploy
```

## 写文章

```bash
npx hexo new "文章标题"
```

文章保存在 `source/_posts/` 目录下，使用 Markdown 格式编写。

## 目录结构

```
source/       # 文章源文件
public/       # 生成的静态文件
themes/       # 主题
scaffolds/    # 文章模板
_config.yml   # 站点配置
```
